ASP.NET CORE 6 MVC

It's an ASP.NET CORE 6 MVC framework web application written in C#, allows users to request permissions, camera recordings and backups from the local IT team.

The web application is designed to make it simple and easy for users to make requests and track the status of their requests.

  • Responsive interface
  • Registration free, works on Intranet
  • User profiles, which include all contact details
  • Claim statuses can be tracked by users and managers
  • Easy to read and understand request form
  • PDF pages and lists with links
  • Global search
  • System settings for configuring services (LDAP, MAIL)
  • Automatic email address notification
  • Automatic data update on Active Directory server
  • Automatic availability monitoring
  • Dedicated administrative interface system
  • Using Microsoft SQL Server, all data displayed is dynamic
  • Dynamic management of parking permissions and phone PINs
  • Built-in error reporting system
  • Active Directory privilege converter
